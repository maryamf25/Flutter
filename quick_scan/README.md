## ğŸš€ Getting Started

### Prerequisites

- Flutter SDK (3.10.4 or higher)
- Android Studio / Xcode
- Physical device (recommended for camera testing)

### Installation

1. **Clone the repository**
   ```bash
   cd quick_scan
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Generate Hive adapter**
   ```bash
   dart run build_runner build
   ```

4. **Run the app**
   ```bash
   flutter run
   ```
and storage permissions are automatically configured in `AndroidManifest.xml`:
```xml
<uses-permission android:name="android.permission.CAMERA" />
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE" />
<uses-permission android:name="android.permission.READ_MEDIA_IMAGES" />
```

## ğŸ› ï¸ Development Notes

### Code Generation
The `scan_item.dart` model uses Hive's `@HiveType` annotations. After any changes to the model:
```bash
dart run build_runner build --delete-conflicting-outputs
```

### Testing
To test camera functionality, use a physical device:
```bash
flutter run -d <device-id>
```

### Troubleshooting
- **Camera not working**: Ensure permissions are granted in device settings
- **Build errors**: Run `flutter clean` then `flutter pub get`
- **Missing .g.dart file**: Run build_runner as shown above
